# TreePlanterMod
木炭と苗木、苗木を使った木材作成レシピの追加

Addition of charcoal, sapling, and wood making recipes using sapling.

## Default ID
- String ID : 28500-28504
- Item ID : 9150,9151
- Recipe ID : 220-222
- GridIndex : 1506,1507
